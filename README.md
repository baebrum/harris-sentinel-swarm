## Overview

**Harris Sentinel Swarm** is a self-aware, coordinated collection of Unmanned Air Vehicles (UAVs) capable of detecting, following, and predicting humans and their movement with and without infared cameras.

**Harris Sentinel Swarm** combines the following objectives:

1. Target Tracking

    Implements machine learning models for UAVs to track and follow moving targets.

2. Target Prediction

    Implements machine learning models for UAVs to track moving targets and predict their future locations.

3. Human detection

    Implements machine learning models for UAVs to detect humans on Infrared camera frames.

4. Collision avoidance simulation

    Implements a collision avoidance system that allows multiple UAVs to navigate safely in an environment.

5. Swarm Coordination

    Implements swarm coordination algorithms for drones, enabling them to perform collective tasks like area coverage, target tracking, or obstacle avoidance.

Throughout the course, the Team will address each objective sequentially. Each completed objective will be serve as a building block for the next objective.
