# Resources

### Milestones

1. Target Tracking
- https://paperswithcode.com/dataset/otb-2013
  - provides object and human tracking
  - could be good if we want drones to track and follow other drone or plane for example
  - source code and dataset included in link

2. Target Prediction
- https://cvgl.stanford.edu/projects/uav_data/
  - provides data from uav
  - diverse object prediction
  - no source code, but dataset included
  - reseearch papers included

3. Human Detection
- https://motchallenge.net/
  - tons of data in various settings
  - cited in research studies mulitple times
  - lots of developer results with source code
